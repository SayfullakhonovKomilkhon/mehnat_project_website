# Project Brief: Mehnat Kodeksiga Sharh

## Overview
**Mehnat Kodeksiga Sharh** (Labor Code Commentary) is an official government portal for the Ministry of Employment and Labor Relations of the Republic of Uzbekistan. The platform provides professional commentary and interpretations of Uzbekistan's Labor Code.

## Core Requirements

### Purpose
- Official commentary platform for Labor Code articles
- Professional legal interpretations and analysis
- AI-powered assistance for labor law questions
- Multi-language support (Uzbek, Russian, English)

### Design Language
- **Aesthetic**: Formal, trustworthy government design
- **Palette**: Official blue-white-gray with gold accents
- **Branding**: State emblem, ministry logos, verification badges
- **Responsiveness**: Mobile-first responsive design

### Technical Stack
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS with custom design system
- **i18n**: next-intl (Uzbek, Russian, English)
- **Animations**: Framer Motion
- **Icons**: Lucide React

### Key Features
1. Landing page with government branding
2. Article commentary browsing and search
3. Individual article detail pages
4. AI chat assistant for questions
5. Full internationalization

## Target Users
- Citizens seeking labor law information
- Employers understanding their obligations
- Legal professionals researching cases
- Government officials referencing codes

## Success Metrics
- Professional government appearance
- WCAG accessibility compliance
- Mobile responsiveness
- Fast page load times
- Clear information architecture
